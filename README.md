# 📊 Power BI Sales Dashboard | Data Analytics Project

## 📌 Project Overview
This Power BI dashboard analyzes retail sales performance, profit trends, customer behavior, and regional insights using the Superstore dataset.

---

## 🎯 Objectives
- Analyze sales and profit trends
- Identify top-performing products
- Compare regional performance
- Track customer contribution
- Build an interactive business dashboard

---

## 🛠 Tools Used
- Power BI
- DAX
- Excel
- Data Visualization

---

## 📊 Dashboard Features
- KPI Cards
- Sales Trend Analysis
- Sales by Category
- Sales by Segment
- Profit by Region
- Top 10 Products
- Interactive Slicers

## 💡 Business Insights
- Technology category generates highest sales
- Consumer segment contributes maximum revenue
- Certain regions show lower profitability
- Top products contribute significant sales volume
